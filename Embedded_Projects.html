Embedded And IOT

  • Automatic Fan Control Based on Temperature

An automatic fan control system based on temperature using a PIC microcontroller monitors environmental temperature via a temperature sensor, such as an LM35. The microcontroller reads the sensor's output and compares it to predefined threshold values. If the temperature exceeds a set limit, the PIC triggers a relay to turn on the fan. When the temperature drops below the threshold, the fan decreases the speed. This system ensures energy-efficient cooling by regulating fan operation based on real-time temperature changes, utilizing a simple control loop.

  • Temperature and Humidity Monitoring System:

A Temperature and Humidity Monitoring System uses sensors like the DHT11 or DHT22 to measure environmental temperature and humidity levels. The data is collected by a PIC microcontroller and displayed on a screen like an LCD or OLED. It can also be transmitted to the cloud or a mobile app for remote monitoring. This system helps monitor climate conditions in homes, greenhouses, or industrial settings, and can trigger alerts or automate actions (e.g., controlling fans or heaters) when specific thresholds are exceeded.
